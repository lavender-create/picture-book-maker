# 読み聞かせ絵本アプリ

子ども向けの **読み聞かせWebアプリ** です。  
文章を画面に表示し、AIによる **音声合成（Text-to-Speech）** を使って自動で読み上げます。  
物語選択から読み終わりまで、子どもが迷わず操作できる流れを意識して作成しました。

---

## 🌟 主な機能
- 複数の物語から選択可能
- 文章を1文ずつ表示しながら進行
- AI音声による自動読み上げ
- 読み聞かせ完了までの画面遷移
- 子ども向けに配慮した文字サイズ・配色・操作性
- 表紙画像のAI生成（任意機能）

---

## 🛠 使用技術
- Python（Flask）
- HTML / CSS / JavaScript
- 音声合成（Text-to-Speech API）
- 画像生成（AI Image Generation）
- GitHub
- Render（デプロイ予定）

---

## 💡 工夫した点
- 子どもが直感的に使えるシンプルなUI設計
- 読みやすさを重視した文字サイズと行間の調整
- 「最後まで読めた」と感じられるおしまい画面の構成
- Flaskを使った複数ページ遷移とAPI連携の実装
- ポートフォリオ掲載を想定したディレクトリ構成の整理
